# Advanced-rider-safety-system
Advanced rider safety system for Rider Safety using IoT, Alcohol Detection, Accident Detection, and Emergency Alert Features.  

## Overview
The Advanced rider safety System is an IoT-based rider safety solution designed to improve road safety and reduce accidents. The system ensures that the rider wears a helmet before starting the vehicle, detects alcohol consumption, monitors accidents, and provides emergency alerts when necessary.

## Objectives
- Improve rider safety.
- Prevent drunk driving.
- Reduce accident-related fatalities.
- Provide emergency notifications during accidents.
- Encourage the use of helmets while riding.

## Features
- Helmet Wearing Detection
- Alcohol Detection
- Accident Detection
- Emergency Alert System
- Buzzer Notification
- Real-Time Monitoring
- Vehicle Ignition Control
- Rider Safety Enhancement

## Components Used
- ESP32 Microcontroller
- Alcohol Sensor
- Helmet Detection Sensor
- GSM Module
- GPS Module
- Buzzer
- Power Supply Unit
- Connecting Wires

## Working Principle
1. The system checks whether the rider is wearing a helmet.
2. Alcohol detection is performed before vehicle ignition.
3. If alcohol is detected, the vehicle ignition is blocked.
4. During riding, the system continuously monitors accident conditions.
5. If an accident occurs, emergency alerts can be sent through GSM communication.
6. The buzzer provides warning indications whenever required.
